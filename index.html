Private Cloud with OpenStack

Openstack is open source and free cloud computing service which provide you deployment base IaaS(Infrastructure as a Service). The tools of openstack is called as "Project" that handled by core cloud computing service like Networking, storage and images service.OpenStack makes it easier and more efficient than than ever to scale and virtualize your world, scale your cloud up or down to meet the demands of your organization.

Here, this my capstone project. I implement core services which is used to create a cloud such as Horizon, Nova-compute, Neutron, Swift, Cinder, Glance, Keystone, Heat etc. By implementing such services, we are able to create an instace of tiny-linux image and running it on my private cloud.

CMS with Node,Express,MongoDB

The Content Management System is allow us to control and manage the digital content of the website. There are various types of CMSes are available, but the major types are ECM(Enterprise Content Management) and WCM(Web Content Mangagement).For the WCM there are good examples like Wordpress,Magento,Wix,joomla,shopify, Drupal etc.

Here, I create a my own basic CSM using Node, Express and MongoDB . The basic functionalities are admin can login add update or delete the pages. User can add and modify their pages. Also I add dynamic header so that header also be modified by admin or user.
